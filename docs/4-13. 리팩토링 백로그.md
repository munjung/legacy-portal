# 4-13. 리팩토링 백로그

[docs/smells.md](./smells.md)의 진단 결과를 [4-12. 우선순위 매트릭스](./4-12.%20우선순위%20매트릭스.md)의 로드맵 순서로 실행 가능한 작업 단위(PR 1개 크기)로 쪼갠 백로그다. 각 항목은 착수 전 [4-10](./4-10.%20도출%20스멜%20-%20심각도%20표.md)의 검증 상태가 합의되어야 하며, 완료 조건(DoD)은 예외 없이 **CLAUDE.md 불변 규칙**(API 엔드포인트·public 메서드 시그니처·요청/응답 형식·DB 저장값·관찰 가능한 동작 보존)과 **특성화 테스트 green 유지**를 포함한다.

## 사용 안내

- 각 항목 착수 전 [4-10](./4-10.%20도출%20스멜%20-%20심각도%20표.md)에서 해당 스멜의 검증 상태가 `☑ 확인됨`인지 확인한다. `☐ 검토 필요` 상태인 항목은 팀 리뷰 없이 그대로 착수하지 않는다.
- 완료 조건(DoD)의 "기존과 동일/불변"은 전부 BL-01에서 확보한 특성화 테스트로 검증한다 — 테스트가 리팩토링 전후로 코드 수정 없이 green이면 완료로 간주한다.
- 공수·순서는 추정치이며, 팀 상황에 따라 [4-12](./4-12.%20우선순위%20매트릭스.md)를 다시 참고해 조정할 수 있다.
